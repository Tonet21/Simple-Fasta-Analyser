# Simple Fasta Analyser

Bioinformatics beginner project.
In this repo I provide a Fasta file with example sequences from roslind.info and a Biopython script that analyses that file.
The script also works with files that only have on sequence. It is writen to provide a summary table (ID, lenght and GC content), a printable template of the compiled statistics (number and percentage of nucleotids, GC content, reverse complement and RNA transcription) and two plots (a histogram for length and one for the GC content).

I am aware that the code is not perfect, but I am learning.
